# Welcome to **The BLK Nest**

Community-vetted, easy-to-use resources for caregivers of Black babies in **Metro Detroit**.  

## About The BLK Nest

We’re building a trusted, searchable repository (and later on, a community hub) so Black families in Metro Detroit don’t have to dig for culturally competent care.  
Your feedback makes these guides better for the next caregiver, thank you for being here.
